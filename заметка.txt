переменная определяется как имя с которым может быть связано значение, буквально это коробка с именем в которое мы помещаем знаение (информацию)

перемення вызывается (обьявляется) ключевым словом LET , после нам необходимо дать имя переменной. 
правила записания имен для переменных;
	переменная может содержать буквы латинмкого алффавита, цыфры и сомволы $, u но при этом первыый символ недолжен быть цыфрой 
	let age;
let info123;
let $size;
let _color; правильная запись

let 123info;
let my-age; неверное значение

именна переменных должны соответсвовать вложеной информации чтобы понятно с че вы работает. если имя состоит из нескольких слов то принято использовать стиль 
	кемел кейс, каждое новое слово пишется с заглавной
	ловер кемел кейс когда каждое новое слово пишется с заглавной но первое с маленькой 
нельзя использовать зарезервированый слова, которые используються самым языком джава скрипт
console.log (myLifeSyle); для проверки работы в консоле пишетсяимя переменной 
обьявлять несколько пременных лучше всего через запятую и с новой строки
let myName = 'Фрилансер по жизни',
	meAge = 36,
	myMessage = 'Живи, а работай в свободное время';
ключевое слово лэт пишеться один раз 
значение переменной можно поменять по ходу дела, положить в нее другую иформацию
USE STRICT - режим который запрещает обьявления переменной без ключевого слова ЛЭТ

обьявления переменных в разных частях джава кода имеет большое значение. это так называемая область видимости переменных 
 переменная нааходящаяся внутри блока инструкций не будет видна за его предалами 

 пременные с одинаковыми именами внутри блока инмтрукуий и за его пределами - это две нзависящие друг от друга переменные 
 function testBlock() {
	let myAge = 36;
	console.log(myAge);
}
testBlock();

let myAge = 18;
console.log(myAge);

 во всем остальном константы такие же как и еремнные, что касается имен и области видимости
 существую Константы которые уже известны до написания программы, их обычно записывают большми буквами (в верхнем регистре) через нижнее подчеркивание (напрмре размер или код цвета) но также сущесвуют конмсты которым задаю значение в ходе создания программ

 способ измениния Клнстатнты несет в себе идею изминения значениявнутри тива информаци, нельзя изменитьсам тип информации, допустим 36, можноизменить на 18 но нельзя изменить на "учим дома"
